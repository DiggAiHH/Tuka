# God Mode Deployment

## Instructions

1. **Analyze requirements**: Deep dive into the current project state and user request.
2. **Install dependencies**: Automatically run `npm install` or `pip install` as needed.
3. **Self-Healing Loop**: If errors occur during any phase, read the error logs, apply a fix, and retry automatically.
4. **Build**: Execute the build process (e.g., `npm run build`).
5. **Start local server**: Launch the development server or local preview.
